step 1 : create a new java prject PublisherSubscriber
step 2 : create a package named pubsub
step 3 : create 2 different classes -> Publisher and Subscriber
step 4 : add you resp code to these 2 classes 
step 5 : now click on project name and click on Build Path
step 6 : click on Configure path and then add external jars 
step 7 : add the jar file :- "activemq-all-5.15.8.jar"
step 8 : open terminal to the folder where you have extracted activemq
step 9 : navigate to bin folder and run command :- "activemq start"
step 10 : Run the Subscriber first 
step 11 : Run the Publisher next 


Download Activemq : https://activemq.apache.org/components/classic/download/
